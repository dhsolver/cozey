# Cozey challenge description

## Tech used

- Database: `SQLite`
- ORM: `Prisma`
- Backend: `Node.js/Express`
- Frontend: `Next.js`
- Language: `Typescript`

## Setup Backend

- `yarn`
- `yarn seed` (This will generate template data, create the database file and insert the seed data.)

- `yarn dev`

## Setup Frontend

Frontend is currently setup to use local backend running at 4040 port, which is the default port configured for the backend.

- `yarn`
- `yarn dev`
